## P0 Critical Issues (Must Fix Before Release)

### PERF-001: GeneratePanel recursive setTimeout — no cleanup on unmount
- **File**: `src/pages/GeneratePanel.tsx:399,403`
- **Impact**: Memory leak + ghost HTTP requests on unmount
- **Fix**: Store timeout ID in `useRef`, clear on unmount via `useEffect` cleanup

### PERF-002: QuickGeneratePanel setInterval + setTimeout leak on unmount
- **File**: `src/pages/QuickGeneratePanel.tsx:110-125`
- **Impact**: Interval and event listener leak when component unmounts during generation
- **Fix**: Track interval/timeout/unwatch in refs, clean up in `useEffect` return

### PERF-003: 13 components use destructured `useAppStore()` — subscribes to ALL 30+ fields
- **Files**: GeneratePanel, EditCanvas, Timeline, BatchPanel, Header, + 8 others
- **Impact**: ~13 unnecessary re-renders per state change across the app
- **Fix**: Refactor to individual selectors: `const x = useAppStore(s => s.x)`

### STATE-001: `deleteCompletedJob` called but never defined — runtime crash
- **File**: `src/components/layout/Timeline.tsx:608,713`
- **Impact**: `TypeError: deleteCompletedJob is not a function` at runtime when user deletes a completed job
- **Fix**: Add `deleteCompletedJob` action to `generationSlice.ts` and `AppState` type

### STATE-002: `assetMetadata` Map never persisted — data loss on reload
- **File**: `src/store/appStore.ts:182-208` (partialize function)
- **Impact**: All asset metadata silently discarded on every page reload
- **Fix**: Add `assetMetadata` to `partialize` with `Array.from(entries())` serialization, reconstruct in `merge`

### STATE-003: No debounce or abort on Generate — rapid clicks fire concurrent jobs
- **File**: `src/pages/GeneratePanel.tsx:199-424`
- **Impact**: Duplicate API calls, state corruption, wasted backend resources
- **Fix**: Add ref-based pending guard or disable button while `isGenerating`

### A11Y-001: No skip-to-content link
- **File**: `src/App.tsx`, `src/components/layout/`
- **Impact**: Keyboard users must tab through entire NavBar to reach main content
- **Fix**: Add visually-hidden skip link as first focusable element

### A11Y-002: Form inputs lack associated labels
- **Files**: `src/components/ui/Input.tsx`, `src/components/ui/Textarea.tsx`
- **Impact**: Screen readers cannot announce input purpose
- **Fix**: Bind `htmlFor`/`id` between labels and inputs

### A11Y-003: Zero `prefers-reduced-motion` support
- **Files**: All components using Framer Motion animations
- **Impact**: Animations cannot be disabled for users with vestibular disorders
- **Fix**: Add `useReducedMotion()` hook and conditionally apply animations

### A11Y-004: Missing `h1` headings on 7 of 8 pages
- **Files**: All panel components
- **Impact**: Screen reader users cannot understand page structure
- **Fix**: Add visually-hidden `h1` to each panel
