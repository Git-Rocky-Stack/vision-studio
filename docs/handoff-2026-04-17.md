## Notes For The Next Agent

Do not commit generated Electron bundle output unless explicitly requested:

```powershell
git restore -- dist-electron/main.mjs dist-electron/preload.cjs
```

The project `npm run dev` starts Vite plus Electron and may also start the Python backend. For browser-only smoke, prefer:

```powershell
npm run build
npx vite preview --host 127.0.0.1 --port 4173 --strictPort
```

Then use Playwright against `http://127.0.0.1:4173`.

If a PowerShell-managed server process leaves child processes behind, stop the port owner:

```powershell
Get-NetTCPConnection -LocalPort 4173 -ErrorAction SilentlyContinue |
  Where-Object { $_.State -eq 'Listen' } |
  ForEach-Object { Stop-Process -Id $_.OwningProcess -Force -ErrorAction SilentlyContinue }
```

When continuing semantic typography cleanup, keep using the same safe sequence:

1. Add one target file to `shellTypographyFiles`.
2. Run `npx vitest run src/styles/ui-glyphs.test.ts --project unit` and confirm it fails only on expected lines.
3. Replace legacy typography with existing semantic utilities.
4. Run the guard, any focused component test, `npm run typecheck`, and `git diff --check`.
5. Commit the small slice.
