## Architecture Notes

### Store Pattern
- Types in `src/store/appStore.types.ts` (`AppState`, `AppActions`)
- Slices in `src/store/slices/` — each exports `initialState` + `create*Actions(set)`
- Combined in `src/store/appStore.ts` with `partialize` for persistence
- Tests in `src/store/appStore.test.ts`
- Component tests use `--project component` (jsdom), unit tests use `--project unit` (node)

### New Navigation Types
- `ActiveTab` now includes `'collections'` (added in Phase 3)
- `GenerateSubMode` now includes `'studio'` (added in Phase 2)
- `ActiveSubMode` will need `WorkflowsSubMode` (`'workflows' | 'pipelines'`) in Task 16

### Key Conventions
- `cn()` from `@/utils/cn` for className merging
- `readFileAsDataUrl` from `@/utils/readFileAsDataUrl` for file reading
- `useShallow` for Zustand selectors in components
- `memo` + named function pattern for component exports
- No non-ASCII characters (glyph policy enforced by tests)
- `afterEach(cleanup)` required in component tests (no `globals: true`)
