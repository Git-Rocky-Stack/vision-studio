  🚀 Build the Installer (3 Options)

  Option 1: One-Click PowerShell (Easiest)

  # In PowerShell, navigate to vision-studio folder
  .\Build-Windows.ps1 -FullBundle

  Option 2: npm Script

  # Full bundle with AI backend
  npm run build:windows:full

  # Frontend only (smaller, faster)
  npm run build:windows

  Option 3: Manual Steps

  # 1. Install dependencies
  npm install

  # 2. Build backend (optional, 30-60 min)
  npm run build:backend

  # 3. Package for Windows
  npm run package:win

  🎨 Before First Build

  1. Create an App Icon (Optional)

  # Option A: Auto-generate
  npm run create:icon

  # Option B: Manual
  # - Create 256x256 image
  # - Convert to ICO at redketchup.io/icon-converter
  # - Save as build/icon.ico

  2. Check Prerequisites

  node --version    # Need v18+
  python --version  # Need 3.10+ (for full bundle)

  # Then sign:
  signtool sign /f cert.pfx /p password `
    /tr http://timestamp.digicert.com `
    /td sha256 /fd sha256 `
    "release\Vision-Studio-Setup-0.1.0.exe"
