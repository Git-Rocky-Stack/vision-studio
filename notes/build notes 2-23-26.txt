  🎯 What Happens Behind the Scenes

  When a user launches the app:

  1. Electron checks for bundled Python executable in resources/
  2. If found: Launches the bundled backend automatically
  3. If not found: Falls back to system Python
  4. First run: Shows setup wizard, downloads models as needed
  5. WebSocket connects for real-time progress updates
  6. User can start generating immediately!

  📊 Bundle Contents

  Bundled Executable Includes:

  • Python 3.11 runtime
  • PyTorch 2.1 with CUDA 12.1
  • Diffusers, Transformers, Accelerate
  • FastAPI, Uvicorn, WebSocket server
  • All Python dependencies from requirements.txt

  • AI Models (FLUX, SDXL, etc.) - 2-24 GB each
  • These download on first use from HuggingFace

  💡 Pro Tips

  1. Reduce bundle size: Edit backend/main.spec excludes list to remove unused PyTorch components
  2. GPU not required: The app works on CPU (just slower)
  3. Models are cached: Downloaded once, reused forever
  4. Offline capable: After first setup, works without internet
